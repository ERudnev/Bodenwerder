# pQRF

**persistent Quantized Relational Form** — SQLite backend for fQSM aspects that declare a retrospection form via `Meta::describe` (`one`/`all` × `field`/`collection`). The engine walks that form to create tables, push quanta/globals/collections, and pull them back into a realm. The aspect-side vocabulary lives in fQSM; this module owns the relational mapping and storage.
