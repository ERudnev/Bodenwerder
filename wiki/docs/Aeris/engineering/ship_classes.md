# DS4 Ship Classification v2

## Базовые классы оборудования и корпусов

Единая шкала используется одновременно для:

* корпусов кораблей
* реакторов
* двигателей
* оружия
* сенсоров
* гироскопов
* накопителей энергии
* прочих крупных систем

Класс определяет занимаемый объём, массу, стоимость и мощность системы.

| Класс | Коэффициент |
| ----- | ----------: |
| A     |           1 |
| B     |           2 |
| C     |           4 |
| D     |           8 |

Классы образуют степень двойки.

---

## Основные классы кораблей

| Корабль | Базовый класс |
| ------- | ------------- |
| Корвет  | A             |
| Фрегат  | B             |
| Крейсер | C             |
| Линкор  | D             |

Класс корабля определяется типовым классом установленного оборудования.

---

## Оружейный бюджет

Вооружение рассматривается как отдельный бюджет.

Для корпуса класса X оружейный бюджет равен двум системным единицам данного класса.

Корвет:

* 2A

Фрегат:

* 4A = 2B

Крейсер:

* 8A = 4B = 2C

Линкор:

* 16A = 8B = 4C = 2D

---

## Размеры оружейных слотов

| Слот | Стоимость |
| ---- | --------: |
| A    |         1 |
| B    |         2 |
| C    |         4 |
| D    |         8 |

Эквивалентности:

* B = 2A
* C = 2B = 4A
* D = 2C = 4B = 8A

---

# Подклассы кораблей

Подкласс определяется способом распределения оружейного бюджета.

Размер корпуса при этом не меняется.

---

## Корветный корпус

### Корвет

Вооружение:

2A

Универсальный малый корабль.

---

### Ударный корвет

Вооружение:

1B

Корвет, вооружённый калибром следующего класса.

---

## Фрегатный корпус

### Эсминец

Вооружение:

4A

Большое количество малых орудий.

---

### Эскорт

Вооружение:

1B + 2A

Смешанная схема вооружения.

---

### Фрегат

Вооружение:

2B

Базовый представитель класса.

---

### Ударный фрегат

Вооружение:

1C

Фрегат, вооружённый калибром следующего класса.

---

## Крейсерный корпус

### Рейдер

Вооружение:

1C + 2B + 2A

Максимальная универсальность вооружения.

---

### Крейсер

Вооружение:

2C (1C + 2B)

Базовый представитель класса.

---

### Тяжёлый фрегат

Вооружение:

2B + 4A

Крейсерский корпус с фрегатной философией вооружения.

---

### Ударный крейсер

Вооружение:

1D

Крейсер, вооружённый калибром следующего класса.

---

## Линкорный корпус

### Большой рейдер

Вооружение:

1D + 1C + 2B + 2A

Максимальный разброс калибров.

---

### Линкор

Вооружение:

1D + 2C

Базовый представитель линкорного класса.

---

### Тяжёлый крейсер

Вооружение:

2C + 4B

Линкорный корпус с крейсерской философией вооружения.

---

### Дредноут

Вооружение:

2D

Ударный линкор.

Концентрация вооружения в главном калибре.
