# Мягкий облом транзакции из worker/action

Нужно добавить механизм мягкого отказа от транзакции из `Writing`/worker-кода, без `throw`.

## Место, где уже нужно заменить временный warning

`modules/raidenmamare.cpp/source/resources/runtimes.q1.cpp`

Сейчас там временно стоит:

- логирование через `base::message(...)`
- `return`

в `resource::Runtimes::Actions::install(...)`, в ветке повторной установки для того же `Device`.

## Что должно быть вместо этого

Не warning в лог, а явный вызов механизма облома транзакции:

- action сообщает причину отказа
- транзакция мягко отклоняется
- изменения не применяются
- без исключения (`throw`)
