# DAQL Workspace
Проекты в workspace могут быть направлены на разработку технологий,
создание игр или комбинацию обоих направлений.

## Методология
`_seed` содержит методику совместной работы с искусственным интеллектом.
Работа начинается с контракта: `_seed/contract.tome`

## Модули
Модули живут в плоской структуре modiles/
могут имет управляющие блоки _management (see _seed/methods/management.tome)
